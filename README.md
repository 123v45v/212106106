## 代码说明

code/model.py：创建并初始化vgg19模型

code/util.py: 对输入的图片进行大小的修改和归一化处理

code/setting.py: 设置图片存储路径以及训练epoch数和学习率等

code/train.py: 用于训练模型

code/app.py: 利用Flask框架调用模型进行使用

templates/index.html：用户使用页面
